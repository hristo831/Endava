# Endava
Endava project is made in C#. It's implement Endava UI and RestFull API tasks.

## Built With
### UI
* C#
* Selenium
* Speckflow
* Nunit
* Autofac container
  
### Service
* C#
* HttpClient
* Nunit

## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/hristo831/Endava.git
   ```
2. Instal Speckflow extention for Visual Studio
